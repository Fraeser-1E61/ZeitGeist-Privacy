Überblick

Diese Datenschutzerklärung gilt für die Android-Apps ZeitGeist und ZeitGeist Pro (nachfolgend „die App"), entwickelt von [Entwicklername].

Alle erfassten Arbeitszeitdaten werden ausschließlich lokal auf deinem Gerät gespeichert. Es gibt keinen eigenen Server, kein Nutzerkonto und keine Cloud-Synchronisation.

Lokale Datenspeicherung

Die App speichert folgende Daten nur auf deinem Gerät:

  • Arbeitszeiteinträge (Check-In/Check-Out, Pausen, Dauer)
  • Eintragstypen (Krank, Urlaub, Feiertag, Frei)
  • Optionale Felder: Projekt, Notiz, Standort
  • App-Einstellungen (Stundensatz, Pausenregeln, Standort-Präferenzen)

Diese Daten verlassen dein Gerät nicht, außer du exportierst sie manuell (PDF, CSV, Excel) und teilst die Datei selbst.

Standortdaten (GPS)

Die App kann optional deinen GPS-Standort ermitteln, wenn du das Standort-Feld in den Einstellungen aktivierst. Opt-In

  • Der GPS-Standort wird nur beim Check-In und beim manuellen Antippen des Standort-Buttons abgefragt
  • Die ermittelten Koordinaten werden über den öffentlichen Dienst Nominatim (OpenStreetMap) in eine lesbare Adresse umgewandelt
  • Dabei werden die Koordinaten einmalig an nominatim.openstreetmap.org übermittelt — eine Drittanbieter-API
  • Der Standort wird als Text im Zeiteintrag gespeichert — nur lokal auf deinem Gerät
Datenschutzerklärung von OpenStreetMap/Nominatim: osmfoundation.org/wiki/Privacy_Policy

Du kannst die Standortfunktion jederzeit in den App-Einstellungen deaktivieren. Das Gerät wird dich um Erlaubnis bitten, bevor der Standort abgerufen wird.

Spritpreise (nur Full-Version)

Die Full-Version der App enthält eine optionale Seite für aktuelle Kraftstoffpreise. Diese Funktion nutzt die Spritpreisportal-API des ADAC / Tankerkönig-Projekts.

  • Es wird eine Postleitzahl oder dein GPS-Standort an die API übermittelt
  • Die API gibt öffentlich verfügbare Kraftstoffpreise zurück
  • Keine personenbezogenen Daten werden dauerhaft dort gespeichert
Diese Funktion ist in ZeitGeist (Free) und ZeitGeist Pro nicht enthalten.

Werbung (nur ZeitGeist Free) Nur Basic

Die kostenlose Version ZeitGeist kann Werbebanner enthalten, die über Google AdMob bereitgestellt werden.

  • Google AdMob kann Gerätekennungen und Nutzungsverhalten auswerten
  • Es kann interessenbezogene Werbung angezeigt werden
  • Du kannst personalisierte Werbung in den Android-Einstellungen unter Google → Datenschutz → Anzeigen deaktivieren
Datenschutzerklärung von Google AdMob: policies.google.com/privacy

ZeitGeist Pro und die Full-Version enthalten keine Werbung.

Berechtigungen

Die App fragt folgende Android-Berechtigungen an:

  • Standort (ACCESS_FINE_LOCATION / COARSE_LOCATION) — Nur wenn Standort-Feature aktiviert. Wird nicht im Hintergrund genutzt.
  • Internetzugang (INTERNET) — Für Nominatim-Adressauflösung, AdMob (Free) und Spritpreise (Full).
  • Speicher (READ/WRITE_EXTERNAL_STORAGE) — Für den Export von PDF, CSV und Excel-Dateien in den Download-Ordner.

Kinder

Die App richtet sich nicht an Kinder unter 16 Jahren. Es werden keine Daten von Kindern wissentlich erfasst.
